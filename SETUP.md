

**Prompt: Build a Professional Investment Advisory Website**

Create a modern, responsive investment advisory website for "Uganda Investment Bridge" using Next.js 14, TypeScript, Tailwind CSS, and Framer Motion. The site should target foreign investors interested in Uganda business opportunities.

**Tech Stack:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- Lucide React for icons

**Key Features:**

1. **Auto-Rotating Hero Slideshow**
   - 5 slides rotating every 40 seconds
   - Slides showcase: General intro, Gold exports, Government contracts, Coffee exports, Land acquisitions
   - Each slide has: badge, title, subtitle, 3 bullet highlights, 2 CTA buttons
   - Background images at 10% opacity
   - Navigation: dots indicator, prev/next arrows, pause on hover
   - Animated progress bar showing slide duration
   - Smooth slide transitions using Framer Motion

2. **Trust Signals Section**
   - 4-column grid: Licensed in Uganda, Independent Advisory, Top-Tier Partnerships, Transaction Success
   - Below that: 6 statistics (8+ opportunities, $50M+ deals, 100% compliance, 2-6 weeks turnaround, 25+ countries, 15 years experience)
   - Icons with animated counters

3. **Core Services Section** (4 cards in responsive grid)
   - Gold Sector Facilitation (amber theme)
   - Government Tender Navigation (indigo theme)
   - Real Estate Acquisition (emerald theme)
   - Contract Implementation & Delivery (blue theme)
   
   Each card includes:
   - Large icon with hover scale effect
   - Title and subtitle
   - "The Problem" paragraph
   - "What We Do" paragraph
   - 4 bullet points with checkmark icons
   - "Learn More" button linking to service detail pages
   - Border color changes on hover

4. **How It Works Section**
   - 3-step process with numbered circles
   - Connecting line between steps
   - Discovery & Planning → Due Diligence → Execution & Delivery
   - Animated on scroll

5. **Why Choose Us Section**
   - 4 cards: Risk Mitigation, Speed & Efficiency, Transparency, End-to-End Support
   - Icon, title, description for each
   - Hover effects

6. **The Diamond Advantage (Comparison)**
   - 2-column layout: Traditional Approach vs Axle Africa Advisory
   - Traditional side: Red theme with X marks
   - Axle side: Green theme with checkmarks
   - 4 comparison points each

7. **Case Studies Section**
   - 3 case study cards in grid
   - Each with: sector badge, title, challenge, solution, results list
   - Hover shadow effects

8. **Testimonials Section**
   - 3 testimonial cards
   - Flag emoji, quote, author name, role, company
   - Border hover effects

9. **Investment Requirements Box**
   - Blue background section
   - 3 feature cards: Local Expertise, Zero Conflict, End-to-End Support
   - White card with: investment minimums for each sector, Uganda Investment Code notice, 2 CTAs

10. **Opportunity Categories**
    - 6 category cards with images
    - Gold, Coffee, Land, Government Contracts, Agriculture, Infrastructure
    - Image header with icon overlay
    - Stats badge on image
    - Hover scale effect on images
    - Links to opportunities page

11. **Final CTA Section**
    - Dark slate background with dot pattern
    - Large heading, description, response time
    - "Get Started" button with arrow

**Design Requirements:**
- Clean, professional B2B aesthetic
- Consistent color scheme: Blue primary, Amber/Gold for mining, Indigo for government, Emerald for real estate
- All sections with scroll animations (fade up, slide in)
- Hover effects on all interactive elements
- Mobile-first responsive design
- Smooth transitions throughout
- High contrast for readability
- Professional typography hierarchy

**Navigation:**
- Sticky navbar with logo and menu items
- Services, About, Resources, Why Uganda, Contact
- Mobile hamburger menu
- Footer with site map, contact info, social links

**Content Tone:**
- Professional, trustworthy, results-oriented
- Focus on risk mitigation and compliance
- Emphasize end-to-end execution (not just advisory)
- Include specific metrics and guarantees
- Address pain points directly

**Animation Guidelines:**
- Fade-in-up on scroll for sections
- Scale on hover for cards
- Slide transitions for carousel
- Smooth color transitions
- Stagger animations for lists
- Progress indicators where applicable

Build this as a complete, production-ready website with clean code, proper TypeScript types, accessibility features (aria-labels), and SEO optimization.
